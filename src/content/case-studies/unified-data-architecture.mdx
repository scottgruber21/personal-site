## The Fragmented Reality
A global retail brand was struggling with "data silos." Marketing data lived in Meta, sales data in Shopify, and support data in Zendesk. Calculating a true Customer Lifetime Value (CLV) was impossible...

## The Architecture
We built a centralized data warehouse using BigQuery. By implementing **dbt (data build tool)**, we transformed raw API extractions into clean, modeled tables ready for visualization...

## The Outcome
With a unified view, the marketing team identified high-value segments that were previously invisible, allowing them to shift budget away from underperforming channels.
